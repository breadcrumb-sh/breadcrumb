---
title: MCP
description: Connect your LLM to Breadcrumb traces via Model Context Protocol.
---

The MCP server lets LLMs query your trace data directly. Connect it to Claude Desktop, Cursor, or any MCP-compatible client.

## Setup

1. Generate an MCP key in your user settings
2. Add the server to your MCP client config:

```json title="claude_desktop_config.json"
{
  "mcpServers": {
    "breadcrumb": {
      "url": "https://app.breadcrumb.sh/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_MCP_KEY"
      }
    }
  }
}
```

## Available tools

| Tool | Description |
|------|-------------|
| `list_projects` | List all projects you have access to |
| `list_traces` | Query traces with filters (status, environment, date range, name) |
| `get_trace` | Get a single trace with all its spans and full content |
| `get_span` | Get a single span with full input/output |
| `list_spans` | Query spans across traces by type, model, provider, status |
| `find_outliers` | Find top N traces by cost, duration, or token usage |
| `introspect_schema` | View the database schema for writing custom queries |
| `run_query` | Run read-only SQL queries against your trace data |

## Access control

MCP keys are scoped to the user who created them. The LLM can only access projects where that user is a member.
