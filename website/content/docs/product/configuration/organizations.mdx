---
title: Organizations
---

Organizations group projects and team members. Each project has its own API keys, traces, and monitor configuration.

Create organizations and projects from the sidebar. Invite team members from organization settings.

## Self-hosted

By default, new signups require an invitation. Set `ALLOW_OPEN_SIGNUP=true` to let anyone create an account. Control who can create organizations with `ALLOW_ORG_CREATION`.
