## Introduction

Breadcrumb is an open-source tracing platform for AI agents and pipelines. It shows each request end to end, from retrieval and tool calls to model generations and final output.

AI systems are hard to debug with logs alone. Breadcrumb turns that flow into a trace you can inspect in one place.

## Why it helps

When something breaks, Breadcrumb helps you see which step failed, what context it received, and what the model returned.
