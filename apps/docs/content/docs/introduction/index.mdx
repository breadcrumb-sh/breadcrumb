Breadcrumb is an open-source tracing platform for LLM applications. It captures every model call your app makes - prompts, completions, tokens, latency, and cost.

<Steps>
  <Step>
    ### Set up hosting

    [Self-host](/docs/setup/self-hosting) Breadcrumb on your own infrastructure, or [sign up for updates](https://tally.so/r/2Er8NL) on the hosted version.
  </Step>
  <Step>
    ### Start tracing

    Follow the [quick start](/docs/introduction/quick-start) to install the SDK and send your first trace.
  </Step>
</Steps>
