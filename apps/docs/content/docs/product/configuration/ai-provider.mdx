The monitor and other AI features require an AI provider to be configured per project.

## Setup

Go to project settings and select a provider:

| Provider | Description |
|----------|-------------|
| **Anthropic** | Claude models |
| **OpenAI** | GPT models |
| **OpenRouter** | Access multiple providers through one API |
| **Custom** | Any OpenAI-compatible endpoint |

Enter your API key and model ID. For custom providers, also set the base URL.
