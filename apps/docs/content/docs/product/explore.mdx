Explore is a chat interface in the dashboard. Ask a question in plain English, get a chart or table back.

## How it works

Type a question like *"Show daily trace volume over the last 30 days"*. The AI writes a SQL query, runs it against your trace data, and renders the result as an interactive chart.

Conversations persist, so you can ask follow-up questions to drill deeper.

## Example questions

- *"What are my top 10 slowest spans?"* - bar chart
- *"Compare token usage across models"* - comparison chart
- *"What's my daily LLM spend this month?"* - cost breakdown
- *"Show error rate by trace name"* - bar chart
- *"Draw a diagram of my most common trace flow"* - mermaid diagram

## Requirements

Explore requires an AI provider to be configured in your project settings.
