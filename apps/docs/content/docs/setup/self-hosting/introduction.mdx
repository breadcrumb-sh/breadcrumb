Breadcrumb is fully open-source and designed to run on your own infrastructure. Your data stays with you - no third-party dependencies, no vendor lock-in.

## Deployment Options

The fastest way to get started is a one-click deploy to [Railway](/docs/setup/self-hosting/railway). It provisions everything - Postgres, ClickHouse, and the Breadcrumb server - in under two minutes.

| Option | Effort | Best for |
| --- | --- | --- |
| [Railway](/docs/setup/self-hosting/railway) | One click | Getting started quickly |
| [Docker Compose](/docs/setup/self-hosting/docker-compose) | ~10 minutes | Running on your own server |
