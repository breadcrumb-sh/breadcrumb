Breadcrumb is an open-source tracing platform for LLM applications. It captures every model call your app makes - prompts, completions, tokens, latency, and cost - and runs an AI agent that monitors your traces for issues that dashboards can't catch.

<Steps>
  <Step>
    ### Set up hosting

    [Self-host](/docs/setup/self-hosting) Breadcrumb on your own infrastructure, or [sign up for updates](https://tally.so/r/2Er8NL) on the hosted version.
  </Step>
  <Step>
    ### Start tracing

    Follow the [quick start](/docs/introduction/quick-start) to install the SDK and send your first trace.
  </Step>
  <Step>
    ### Monitor your agents

    Configure an [AI provider](/docs/product/configuration/ai-provider) and the [monitor](/docs/introduction/monitoring) starts watching your traces automatically.
  </Step>
</Steps>
