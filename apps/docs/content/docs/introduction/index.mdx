Your agents can hallucinate, lose context, and drift from their goal - all while returning a 200. Breadcrumb traces every LLM call and monitors your traces with an AI agent that catches what dashboards can't.

Open source. Self-hosted. [Get started in minutes](/docs/introduction/quick-start).

![Breadcrumb dashboard](/docs/homepage.webp)

## Setup

<Steps>
  <Step>
    ### Deploy Breadcrumb

    [Railway](/docs/setup/self-hosting/railway) (one click) or [Docker Compose](/docs/setup/self-hosting/docker-compose) (your server). [Hosted version](https://tally.so/r/A7xjRB) coming soon.
  </Step>
  <Step>
    ### Add the SDK

    Works with [Vercel AI SDK](/docs/sdks/ai-sdk) or the [TypeScript SDK](/docs/sdks/typescript/installation) for manual instrumentation.
  </Step>
  <Step>
    ### Enable monitoring

    Set an [AI provider](/docs/product/configuration/ai-provider). The [monitor](/docs/introduction/monitoring) scans traces and creates tickets when it finds problems.
  </Step>
</Steps>
